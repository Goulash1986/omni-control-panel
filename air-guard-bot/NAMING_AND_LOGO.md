## Що робити коли визначишся

1. **Згенеруй логотип** через Midjourney (платно) або DALL-E 3 (через ChatGPT Plus) за prompt-ом обраного варіанту
2. Згенеруй 4 варіанти, обери найкращий
3. У Figma або Photoroom: обрізати у круг (Telegram автоматично кропить, але краще завчасно)
4. Зберегти PNG 640×640
5. Відправити `@BotFather` → команда `/setuserpic` для свого бота
6. Описати бот через `/setdescription`:
   ```
   Персональний помічник з повітряної безпеки для Дніпра.
   Real-time моніторинг, мапи загроз, дзвінок при IMMEDIATE.
   Цей бот приватний.
   ```
7. Команди через `/setcommands`:
   ```
   start - Початок роботи
   whoami - Хто я зараз
   status - Поточна обстановка
   sos - Найближче укриття
   safe - Я в безпеці
   pause - Тимчасово приглушити
   digest - Звіт за день
   ```
