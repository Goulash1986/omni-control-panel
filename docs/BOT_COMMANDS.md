# Команды бота

## Команды (через `/`)

| Команда | Что делает |
|---|---|
| `/start` | Запуск + кнопка-вход в Mini App. Первый раз — назначает тебя owner |
| `/start auth_<TOKEN>` | Подтверждает вход в браузер (этот URL формируется автоматически при QR) |
| `/panel` | Кнопка «Открыть панель» |
| `/today` | Сводка за сегодня (доход/расход/привычки) |
| `/week` | Сводка за текущую неделю |
| `/month` | Сводка за текущий месяц |
| `/inbox` | Топ-8 непрочитанных сообщений |
| `/add` | Меню «Что добавляем?» (расход / доход / привычка / идея / задача / пост) |
| `/done <привычка>` | Отметить привычку выполненной (`/done спорт`). Если привычки ещё нет — создаёт |
| `/ask <вопрос>` | LLM-аналитика по твоим данным (`/ask Куда уходят деньги?`) |
| `/cancel` | Отменить текущий шаг диалога |
| `/id` | Твой Telegram ID |
| `/help` | Шпаргалка |

## Свободный текст

Бот сам понимает (через regex + GPT, если есть `OPENAI_KEY`):

| Сообщение | Действие |
|---|---|
| `500 кофе` | Расход 500 в категорию «кофе» |
| `+500 кофе` | То же (минус по умолчанию) |
| `-500 такси` | Расход |
| `+5000 зарплата` | Доход |
| `200₴ обед`, `200 грн обед`, `$10 кофе` | Распознаёт валюту |
| `сделал спорт`, `тренировка ✅` | Отметить привычку |
| `идея: купить новый ноут` | В инбокс как идея |
| Что-то непонятное | Падает в инбокс «непрочитанные», увидишь в панели |

## Голосовые

Любое голосовое → Whisper расшифровывает → дальше как обычный текст.
Перед расшифровкой бот пишет «🎙 Расшифровываю...», потом показывает текст и результат.

Без `OPENAI_KEY` голос не работает — бот скажет об этом честно.

## Фото

Если в чате с ботом отправить фото и в этот момент идёт диалог про «новый пост» —
картинка станет постером поста и привяжется к нему. В остальных случаях просто сохранится
в Drive и бот пришлёт ссылку.

## Inline-кнопки

`/add` показывает кнопки. Тап → бот переключает тебя в режим ожидания нужного типа ввода.

## Уведомления

Бот сам пишет тебе:
- 9:00 — утренний обзор привычек
- 21:00 — итог дня
- Понедельник 10:00 — итог недели
- При активации рекламы в панели
- За час до автоматического удаления рекламы (`del_after`)

Можно отключить триггеры в Apps Script → Triggers → 🗑.
